*******************************************************************************
Setup COMPAS Viewer
*******************************************************************************

:doc:`COMPAS VIEWER<compas_viewer:index>` is a standalone CAD viewer and data
management interface for COMPAS. It is based on Qt (through PySide6) and OpenGL
(through PyOpenGL) and can be installed on Windows, OSX, and Linux to work with
COMPAS geometry and data objects without further requirements or additional software.

The COMPAS Viewer can be used to visualize :ref:`robot_cell`
with their RobotModel, ToolModels and RigidBodyModels. 
It is also possible to add simple user interfaces to the COMPAS Viewer and interact
with the objects, such as chaning the robot :ref:`configuration` 
and moving object in the scene. 

Please follow the installation instructions in the
:doc:`COMPAS VIEWER documentation<compas_viewer:installation>` to install the viewer.
