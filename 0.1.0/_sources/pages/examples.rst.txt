********************************************************************************
Examples
********************************************************************************

There are multiple ways to run the examples below. Most can be executed from
any python environment, for instance:

  * An interactive ``python`` session.
  * An IDE or text editor (e.g. Eclipse, PyCharm, Sublime Text, etc)
  * Inside CAD software (e.g. Rhino, Grasshopper, Blender).
